Cloudbees Infrastructure

All following URLs are authenticated and can only be accessed by Cloudbees users which are part of the project's team.

  • Jenkins instance; you can control and schedule Maven builds (see Jenkins configuration for more details) https://[account_id].ci.cloudbees.com
  • Maven Releases Repo; WebDAV drive which (optionally) contains all build artifacts which are released; no snapshots can be deployed on this repo https://repository-[account_id].forge.cloudbees.com/release
  • Maven Snapshots Repo; WebDAV drive which (optionally) contains all build artifacts which are continuously deployed (*-SNAPSHOT) https://repository-[account_id].forge.cloudbees.com/snapshot
  • Maven Private Repo; WebDAV drive which (optionally) contains the Maven-generated site and Jenkins configurations, such as Maven settings.xml. Site currently does not show since no mime-type text/html is applied to *.html; if would be very cool if Cloudbees could add this feature. https://repository-[account_id].forge.cloudbees.com/snapshot
  • Git/Svn Code Repositories; we're currently using the }Sourcesense gitHub account} for hosting this project code (and this Maven site), in order to have it publicly available, but this sample has been widely tested also using Git on Cloudbees. {{{https://maoo.forge.cloudbees.com/projects/testme/show}https://[account_id].forge.cloudbees.com/projects/[code_repository_name]/show
  • Staging; it hosts the web application deployed via Cloudbees client API http://application_id.account_id.cloudbees.net
