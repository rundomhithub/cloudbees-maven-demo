Welcome to Cloudbees Maven Demo!

Cloudbees Maven Demo is a porting of the default HelloWorld example - which is shipped with CloudBees free PaaS offer - to Apache Maven. It offers a collection of tips & tricks to achieve a simple J2EE demo hosted by CloubBees just using Apache Maven; no Cloudbees SDK is needed, no Ant scripts.

This effort is targeted to people who want to start playing with Cloudbees using Apache Maven. Please also check the official Cloudbees Maven guide
